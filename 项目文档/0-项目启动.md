# 项目部署与启动

## 1. 环境要求

- Python 3.11+
- Node.js 18+（Web UI 用 pnpm）
- PostgreSQL
- Docker（可选，跑 Qdrant 向量库）

## 2. 依赖服务

**PostgreSQL** — 创建数据库 `hivemindos`，记下连接信息。

**Qdrant**（智慧语义检索，可选）：

```bash
docker run -p 6333:6333 qdrant/qdrant
```

## 3. 一键安装（推荐）

```bash
./scripts/install.sh              # 完整安装
./scripts/install.sh --with-qdrant  # 附带启动 Qdrant 容器
./scripts/start.sh                # 启动后端 + 前端
```

## 4. 后端（Knowledge Base API）

```bash
# 项目根目录
cp .env.example .env          # 填入 OPENAI_API_KEY、DATABASE_URL 等
pip install -r requirements.txt
python scripts/migrate_db.py  # 初始化 / 升级数据库
uvicorn memory_layer.knowledge_base.app.main:app --reload --port 8006
```

- API 文档：http://localhost:8006/docs
- 健康检查：http://localhost:8006/health

## 5. 前端（Web UI）

```bash
cd webui
cp .env.example .env          # 填入 DATABASE_URL、AUTH_SECRET 等
pnpm install
pnpm db:push                  # 同步 Prisma 表结构
pnpm dev
```

- 访问：http://localhost:3000

## 6. 启动顺序

```
PostgreSQL → Qdrant（可选）→ 后端 :8006 → 前端 :3000
```

## 7. 运维命令

```bash
python scripts/list_prompts.py    # 查看全部提示词
python scripts/list_settings.py   # 查看全部业务配置
python scripts/migrate_db.py --list   # 查看数据库迁移状态
python scripts/sync_memory_vectors.py # 同步智慧向量（改 embedding 后执行）
```
