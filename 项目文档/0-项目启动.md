# 项目部署与启动

## 1. 环境要求

- Python 3.11+
- Node.js 18+（Web UI 用 pnpm）
- PostgreSQL
- Docker（可选，跑 Qdrant 向量库）

## 2. 依赖服务

**PostgreSQL** — 创建数据库 `hivemindos`，记下连接信息。

**Qdrant**（智慧语义检索，可选）：

```bash
docker run -p 6333:6333 qdrant/qdrant
```

## 3. 一键安装（推荐）

```bash
./backend/scripts/install.sh              # 完整安装
./backend/scripts/install.sh --with-qdrant  # 附带启动 Qdrant 容器
./backend/scripts/start.sh                # 启动后端 + 前端
```

> 仓库已按 **backend + frontend 物理分仓**：后端 Python 全在 `backend/`，前端在 `webui/`。
> 后端命令统一在 `backend/` 目录下执行（脚本会自动处理路径）。

## 4. 后端（HiveMindOS API）

```bash
# 项目根目录：.env 仍放在根目录（后端会向上读取 ./.env 与 ./webui/.env）
cp .env.example .env          # 填入 OPENAI_API_KEY、DATABASE_URL 等
cd backend
pip install -r requirements.txt
python scripts/migrate_db.py  # 初始化 / 升级数据库
lsof -t -i:8006 | xargs kill -9 ; clear && uvicorn main:app --reload --port 8006
```

- API 文档：http://localhost:8006/docs
- 健康检查：http://localhost:8006/health

## 5. 前端（Web UI）

```bash
cd webui
cp .env.example .env          # 填入 DATABASE_URL、AUTH_SECRET 等
pnpm install
pnpm db:push                  # 同步 Prisma 表结构
lsof -t -i:9000 | xargs kill -9   &&   clear &&   pnpm run dev -- port 9000
```

- 访问：http://localhost:9000

## 6. 启动顺序

```
PostgreSQL → Qdrant（可选）→ 后端 :8006 → 前端 :9000
```

## 7. 运维命令

```bash
# 均在 backend/ 目录下执行：cd backend
python scripts/list_prompts.py    # 查看全部提示词
python scripts/list_settings.py   # 查看全部业务配置
python scripts/migrate_db.py --list   # 查看数据库迁移状态
python scripts/sync_memory_vectors.py # 同步智慧向量（改 embedding 后执行）
python scripts/seed_demo_ui_samples.py  # 灌入 Skills / Playbook / 演示智慧（见 10-Org-Playbook.md）
```
