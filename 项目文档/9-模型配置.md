# 模型配置

> 注册表：`model_layer/settings/models.yaml`  
> 加载代码：`model_layer/registry.py`  
> 调用门面：`model_layer/client.py`

---

## 一、策略

**OpenAI 为主模型**：`default` / `fast` / `chat_tools` / `embedding` 默认均走 OpenAI。

**其他 provider 按需使用**：在 `models.yaml` 增加 profile，在 `prompts.yaml` 对应节点写 `model: <profile_id>` 即可，无需改代码。

| Profile | Provider | 典型用途 |
|---------|----------|----------|
| `default` | openai | 规划主持、FinalReflect、强推理 |
| `fast` | openai | L1/L2、Lint、Reflect、Query |
| `chat_tools` | openai | Chat 工具搜集 |
| `embedding` | openai | 智慧 / 经验向量 |
| `anthropic_strong` | anthropic | 可选，特定 prompt 指定 |
| `anthropic_fast` | anthropic | 可选，特定 prompt 指定 |

---

## 二、职责边界

| 层次 | 配置文件 / 模块 | 谁维护 | 定义什么 |
|------|-----------------|--------|----------|
| **密钥** | `.env` → `OPENAI_API_KEY` / `ANTHROPIC_API_KEY` | 运维 | 各 Provider 独立密钥 |
| **模型注册表** | `model_layer/settings/models.yaml` | 平台工程 | profile → provider + model + dim |
| **节点用哪档** | `prompts.yaml` 的 `model: fast \| default \| anthropic_strong …` | 平台工程 | 引用 profile id |
| **怎么说** | `prompts.yaml` system / user 模板 | 平台工程 | 与模型解耦 |

**原则：** yaml 定「连哪个 provider、用哪个 model」；prompts 定「用哪个 profile」；env 可选覆盖 model id，**不改 provider**。

---

## 三、Profile 结构（当前默认）

```yaml
profiles:
  default:
    kind: chat
    provider: openai
    model: gpt-4o

  fast:
    kind: chat
    provider: openai
    model: gpt-4o-mini

  chat_tools:
    kind: chat
    provider: openai
    model: gpt-4o

  embedding:
    kind: embed
    provider: openai
    model: text-embedding-3-small
    dim: 1536

  anthropic_strong:    # optional: true — 无 API key 时不报启动 warn
    kind: chat
    provider: anthropic
    model: claude-opus-4-7
    optional: true
```

---

## 四、在特定节点使用其他模型

只需两步：

1. 在 `models.yaml` 增加 profile（或启用已有 `anthropic_*`）
2. 在 `prompts.yaml` 改对应节点的 `model` 字段

```yaml
# prompts.yaml
agents.planning_chair:
  model: anthropic_strong   # 仅此节点走 Anthropic
  system: |
    ...
```

未配置 `ANTHROPIC_API_KEY` 时，启动会 **warn**，但不影响 OpenAI 主路径。

```env
FAST_MODEL=gpt-4o-mini
EMBEDDING_MODEL=text-embedding-3-small

# 仅在使用 anthropic_* profile 时需要
# ANTHROPIC_API_KEY=sk-ant-...
```

---

## 六、更换 Embedding

1. 编辑 `models.yaml` → `profiles.embedding`（provider / model / **dim**）
2. 若 `dim` 变化，重建向量：`python scripts/sync_memory_vectors.py`
3. 重启或 `reload_registry()`
