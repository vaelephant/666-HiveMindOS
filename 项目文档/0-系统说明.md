 HiveMind OS 一个工程级的目录结构。
目标是：
	•	清晰分层
	•	未来可扩展
	•	可以复用 ZeroClaw
	•	支持 Router / Memory / Agents / Flow

我给你设计的是一个 AI Operating System 架构级目录，不是普通项目结构。
