## 记录范围

| 分类 | 触发点 | 示例 action |
|------|--------|-------------|
| **task** | 自主任务 Executor 每步执行 | `task.llm_generate`、`task.search_wiki` |
| **wiki** | 候选编译进 Wiki | `wiki.compile` |
| **candidate** | 人工批准 / 驳回候选 | `candidate.approve`、`candidate.reject` |

| **deliverable** | 任务保存交付物 | `task.save_deliverable` |
| **lint** | Wiki 质量巡检自动化 | `wiki.lint` |
| **automation** | 工作流 / 单步自动化 | `workflow.run` |

失败步骤记 `status=error`，不阻断主流程。

## API

Base：`http://localhost:8006/api/v1`（WebUI 经 `/api/kb/{orgId}/audit/events` 代理）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/orgs/{org}/audit/events` | 事件列表 + 统计摘要 |
| GET | `/orgs/{org}/audit/export` | 导出 CSV / JSON |
| GET | `/orgs/{org}/audit/stats` | 仅统计 |

查询参数：`days`（1–365）、`category`、`action`、`status`、`q`（全文搜索）、`limit`、`offset`、`user_id`

```bash
curl "http://localhost:8006/api/v1/orgs/demo/audit/events?days=7&category=task&q=Wiki"
curl "http://localhost:8006/api/v1/orgs/demo/audit/export?days=30&format=csv" -o audit.csv
```
