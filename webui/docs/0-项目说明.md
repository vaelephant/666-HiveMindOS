# 项目名称

**模型工厂和数据工厂**

面向大模型与多模态数据的 **AI 中台**（参考界面：**星海·AI中台**）：覆盖从数据采集与标注、训练与评测到模型发布与推理服务的全链路；产品形态上可包含平台首页、能力全景（全息）、中台能力与模型市场等入口。

## 1. 数据工厂

数据工厂负责 **数据资产的全生命周期管理**，支撑高质量训练与评测数据的持续产出。

### 1.1 核心能力概览

| 模块 | 说明 |
|------|------|
| **数据工坊** | 数据加工、流水线或协作式数据处理相关能力（与业务编排衔接） |
| **数据中心** | 数据集的汇聚、目录、版本与基础治理 |
| **标注中心** | 标注场景、标注项目、个人任务与标签模板；连接「标什么」与「谁来标」 |
| **存储中心** | 对象与文件的统一存储与访问，支撑大规模数据与产物归档 |

### 1.2 标注中心：多模态标注场景

标注场景按 **数据模态** 组织，包括但不限于：**图片、大模型对话、音频、NLP、视频、点云** 等，便于团队按任务类型选用模板与工具链。

**大模型对话** 方向典型场景示例（可与截图产品对齐）：

- **多轮对话** — 多轮用户与助手往来的对话标注  
- **单轮对话** — 单轮问答对标注  
- **大模型文档标注** — 文档级片段、要点或结构标注  
- **行业数据标注** — 垂直领域（如医疗、法律、金融等）专用规范与任务  
- **黄金模板** — 高质量标准样例或范式，用于对齐数据规格  
- **问答生成** — 由给定材料生成问答对  
- **单轮长文理解** — 单轮场景下的长文本理解与标注  
- **大模型打分标注** — 对模型回复进行质量、安全性等维度评分  
- **学科试卷拆分** — 试题卷面结构化拆分（题块、选项、答案等）  
- **图文信息标注** — 图像与文本的对应关系或联合标注  
- **大模型指令注释** — 指令（Prompt）的意图、约束与改写类标注  
- **思维链标注** — 逐步推理过程（CoT）的显式标注与校对  

每个场景通常支持 **体验**（快速试用流程）与 **创建**（落为新项目/任务）。

## 2. 模型工厂

模型工厂负责 **从训练到上线推理** 的模型侧能力，与数据工厂通过数据集、评测集与标注规范联动。

### 2.1 核心能力概览

| 模块 | 说明 |
|------|------|
| **训练中心** | 训练任务编排、资源调度、实验与产出管理 |
| **模型中心** | 模型版本、制品（权重、配置、卡片）与生命周期 |
| **评测中心** | 自动化/人工相结合的评测任务、指标与报告 |
| **推理服务** | 模型部署、服务路由、扩缩容与在线调用 |
| **镜像中心** | 训练与推理环境镜像的构建、管理与分发 |
| **资源中心** | 计算、卡时、队列配额等平台资源统一视图与管理 |

### 2.2 与数据工厂的协作关系

- 训练与评测依赖 **数据中心** 的数据集与 **标注中心** 产出的标签数据。  
- 评测体系可与 **大模型打分**、**多轮/单轮对话** 等标注场景共用规范，形成闭环。  
- **推理服务** 消费 **模型中心** 发布的版本，并可通过运营数据反哺数据与标注需求。
