# 导航与路由 — 开发参考

本文档与产品左侧菜单一致，并给出 **开发用标识（`navKey`）**、**建议 URL 前缀** 与 **场景枚举**，便于实现侧栏、权限、埋点与 App Router 目录对齐。  
**说明**：路径为约定草案；若已存在线上路由规范，以实现为准并回写本文档。

---

## 1. 顶层菜单（主导航）

| `navKey` | 菜单文案 | 建议路径 | 归属（产品域） |
|----------|----------|----------|----------------|
| `home` | 首页 | `/` | 通用 |
| `data_workshop` | 数据工坊 | `/data/workshop` | 数据工厂 |
| `data_center` | 数据中心 | `/data/center` | 数据工厂 |
| `annotation` | 标注中心 | （父级，见 §2） | 数据工厂 |
| `training` | 训练中心 | `/training` | 模型工厂 |
| `models` | 模型中心 | `/models` | 模型工厂 |
| `evaluation` | 评测中心 | `/evaluation` | 模型工厂 |
| `inference` | 推理服务 | `/inference` | 模型工厂 |
| `platform` | 平台管理 | `/platform` | 平台治理 |
| `registry` | 镜像中心 | `/registry` | 模型工厂 / 运维 |
| `resources` | 资源中心 | `/resources` | 模型工厂 / 运维 |
| `storage` | 存储中心 | `/storage` | 数据工厂 |

> **实现说明（本仓库）**：营销落地页占用 `/`，平台「首页」实际路由为 `/home`，菜单数据见 `src/config/navigation.ts` 中的 `PLATFORM_HOME_PATH`。

可选：在代码中导出为常量数组以保持菜单顺序与产品一致。

```ts
// 示例：主导航顺序（仅作结构参考）
export const PRIMARY_NAV_ORDER = [
  "home",
  "data_workshop",
  "data_center",
  "annotation",
  "training",
  "models",
  "evaluation",
  "inference",
  "platform",
  "registry",
  "resources",
  "storage",
] as const;
```

## 3. 标注场景 — 分类侧栏（二级）

进入 **标注场景** 后，左侧 **按数据类型** 筛选；建议用 **路径段** 或 **查询参数** 二选一，团队统一即可。

### 3.1 分类枚举（`sceneCategory`）

| `sceneCategory` | 菜单文案 | 建议 URL 段 | 建议 query（若不用段） |
|-----------------|----------|-------------|-------------------------|
| `image` | 图片 | `image` | `?category=image` |
| `llm_dialogue` | 大模型对话 | `llm-dialogue` | `?category=llm_dialogue` |
| `audio` | 音频 | `audio` | `?category=audio` |
| `nlp` | NLP | `nlp` | `?category=nlp` |
| `video` | 视频 | `video` | `?category=video` |
| `point_cloud` | 点云 | `point-cloud` | `?category=point_cloud` |

**路径示例**

- `/annotation/scenes/llm-dialogue` — 推荐：分类 + 列表同页，利于分享链接。
- 或 `/annotation/scenes?category=llm_dialogue`。

### 3.2 大模型对话 — 场景卡片（`sceneType`）

用于卡片跳转「体验 / 创建」、埋点、配置下发 ID；名称与产品文案对齐，标识用 **蛇形英文**。

| `sceneType` | 产品文案 |
|-------------|----------|
| `multi_turn_dialogue` | 多轮对话 |
| `single_turn_dialogue` | 单轮对话 |
| `llm_document_annotation` | 大模型文档标注 |
| `industry_data_annotation` | 行业数据标注 |
| `golden_template` | 黄金模板 |
| `qa_generation` | 问答生成 |
| `single_turn_long_text` | 单轮长文理解 |
| `llm_scoring` | 大模型打分标注 |
| `exam_paper_split` | 学科试卷拆分 |
| `image_text_annotation` | 图文信息标注 |
| `instruction_annotation` | 大模型指令注释 |
| `chain_of_thought` | 思维链标注 |

**详情 / 创建页路径示例**

- `/annotation/scenes/llm-dialogue/multi-turn-dialogue`
- `/annotation/scenes/llm-dialogue/new?type=multi_turn_dialogue`（创建向导）

具体以产品设计（是分页详情还是弹窗）为准。
