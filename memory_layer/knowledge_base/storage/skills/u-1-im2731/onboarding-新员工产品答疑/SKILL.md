# 新员工 HiveMind 产品答疑（Chat / 智慧 / 任务）

## 适用场景

- 任务类型：`onboarding`
- 新员工询问「Chat 和智慧进化有什么区别」「任务中心干什么」
- 内部培训或 HR 带新人熟悉平台

## 推荐步骤

1. query_wiki — 读取产品架构与八层说明
2. 按三层记忆模型解释：Chat 即时问答 → L1/L2 智慧沉淀 → 自主任务引擎带 Rubric 执行
3. 给出 3 个可动手练习：发一条 Chat、在智慧进化页看提取结果、创建一个简单自主任务
4. 指向 Playbook 与 Skills 页面说明组织级约定

## 反思要点

- 不要一次讲完整八层架构，先给「今天能用的三件事」
- 强调企微通道与 Web Chat 共用同一套智慧与 Playbook

## 产出示例

Chat 是「问一句答一句」，会引用 Wiki 和你的长期智慧；智慧进化是 Chat 之后自动提炼的 preference / project / decision；任务中心里的自主任务适合「帮我完成一个业务目标」，会规划、执行、反思并沉淀 Skill。建议你先绑定企微，在 Playbook 里确认团队语气约定。
